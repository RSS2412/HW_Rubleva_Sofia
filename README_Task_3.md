Описание онтологии
В данном задании описана онтология, содержащая в себе знания о классификации дефектов стеклотары. В онтологии реализовано SWRL-правило импликации, которое с помощью логического вычислителя (Drools+reasoner) относит деффекты к классу критический дефект при выполнении определенных уловий.
